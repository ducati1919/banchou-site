見学・お申込み
Q
見学は予約が必要ですか？
予約なしでもお越しいただけます。ただし、事前にお電話またはLINEでご連絡いただくと、スムーズにご案内できます。
Q
見学は無料ですか？
完全無料です。実際のお部屋のサイズ感や雰囲気をご確認いただいてからご契約いただけます。お気軽にお越しください。
Q
申込みに必要なものは何ですか？
本人確認書類（運転免許証・健康保険証・マイナンバーカードなど）と初月の賃料のみです。印鑑は不要です。
Q
契約当日から荷物を入れられますか？
はい、ご契約手続き完了後、その日すぐにご利用いただけます。最短即日でのご利用開始が可能です。
Q
希望の部屋が空いていない場合はどうなりますか？
空き待ちのご登録が可能です。空きが出次第ご連絡いたします。また、ご希望のサイズに近い別のお部屋をご提案することもできますので、お気軽にご相談ください。
Q
法人での契約は可能ですか？
はい、法人でのご利用も歓迎しております。在庫保管・書類保管・季節商品の保管などにご利用いただいています。詳しくはお問い合わせください。
💰
料金・費用
Q
初期費用はいくらかかりますか？
初期費用は¥0です。敷金・礼金・保証金・事務手数料など一切かかりません。ご利用開始月の賃料のみご準備ください。
Q
月額料金はいくらですか？
お部屋のサイズにより異なります。最低2,700円（税込）〜で、4.4帖の最大サイズで39,600円（税込）です。詳しくは料金ページまたはお部屋一覧をご覧ください。
Q
支払い方法はどうなっていますか？
銀行振込（月払い）にてお願いしております。詳しくはご契約時にご案内いたします。
Q
途中で部屋のサイズを変更できますか？
空き状況によっては別のお部屋への変更（アップサイズ・ダウンサイズ）が可能です。ご希望の場合はお気軽にご相談ください。
Q
退去時にクリーニング費用などはかかりますか？
退去費用は¥0です。原状回復費用・クリーニング費用は一切かかりません。荷物をお出しいただければ手続き完了です。
🔑
ご利用方法
Q
何時でも荷物を出し入れできますか？
はい、24時間365日、いつでも出し入れ自由です。深夜・早朝・祝日もご利用いただけます。
Q
入室はどうやってするのですか？
ご契約時にお渡しする暗証番号で電子ロックを解錠してご入室いただきます。鍵の紛失の心配がなく、番号は安全のため定期的に変更も可能です。
Q
台車は使えますか？
台車を無料で貸し出ししています。重い荷物の搬入・搬出もラクラクです。
Q
車で荷物を運べますか？
はい、駐車スペースを完備しています。お車での搬入・搬出も安心してご利用いただけます。
Q
何を保管できますか？保管できないものはありますか？
家具・家電・衣類・書類・趣味グッズ・季節用品など幅広くご利用いただけます。
ただし、以下のものは保管できません：
危険物・爆発物・引火物・薬品・食品・生き物・法律に触れるもの・悪臭を発するもの・他者の財産など。
Q
家族や友人も入室できますか？
暗証番号をお伝えすれば、ご家族やご友人もご利用いただけます。ただし、第三者への暗証番号の無断提供はご遠慮ください。
🛡️
セキュリティ
Q
防犯対策はどうなっていますか？
防犯カメラを館内各所に設置しています。また、入口は電子ロックで施錠されており、契約者のみが入室可能です。
Q
女性一人でも安心して利用できますか？
はい、女性のお客様にも多くご利用いただいています。館内は明るく清潔で、防犯カメラも設置済みです。夜間・早朝の利用も安心してご利用いただけます。
Q
荷物に保険はかけられますか？
弊社では保険の取り扱いは行っておりません。貴重品や高価な品物を保管される場合は、お客様ご自身で保険にご加入されることをおすすめします。
📋
解約・変更
Q
解約はいつでもできますか？
はい、いつでも解約できます。解約希望月の1ヶ月前までにお電話またはLINEにてご連絡ください。
Q
解約時にかかる費用はありますか？
退去費用は¥0です。原状回復費用・クリーニング費用など一切かかりません。荷物をお出しいただければ完了です。
Q
最低利用期間はありますか？
特に最低利用期間の定めはありません。ただし、解約は1ヶ月前のご連絡をお願いしておりますので、ご了承ください。
Q
契約者の変更（名義変更）はできますか？
名義変更はお電話またはLINEにてご相談ください。必要書類などについて個別にご案内いたします。
